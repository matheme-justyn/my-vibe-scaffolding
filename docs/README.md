# Project Documentation

這個目錄用來存放**你的專案**的文件。

## Template 文件

Template 的文件都在 `.scaffolding/docs/`：
- 模組系統指南
- 開發流程規範
- 工具使用說明
- Template ADRs

## 建議的專案文件結構

```
docs/
├── README.md          # 這個檔案
├── adr/               # 專案的架構決策記錄
│   ├── 0100-*.md     # 你的 ADR (從 0100 開始)
│   └── README.md
├── api/               # API 文件
├── guides/            # 使用指南
└── architecture/      # 架構圖、說明
```

參考 `.scaffolding/docs/DOCUMENTATION_GUIDELINES.md`
