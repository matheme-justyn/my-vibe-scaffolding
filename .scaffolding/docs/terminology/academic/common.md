## Notes

### Context-Specific Translations

**「研究」(Research / Study)**:
- 兩者可互換，Research 較正式，Study 較口語
- Context: Research methodology (研究方法論)

**「框架」(Framework)**:
- 🇹🇼 TW: 「理論框架」（學術語境）vs「框架」（軟體語境）
- 🇨🇳 CN: 「理论框架」（學術）vs「框架」（軟體）
- Context: Theoretical framework (理論框架) in academic writing

**「資料」vs「數據」(Data)**:
- 🇹🇼 TW: 使用「資料」（通用）
- 🇨🇳 CN: 使用「数据」（通用）
- Context: Research data (研究資料/数据)

**「質性」vs「質的」(Qualitative)**:
- 🇹🇼 TW: 使用「質性」
- 🇨🇳 CN: 使用「质性」
- 🇯🇵 JP: 使用「質的」

**「量化」vs「量的」(Quantitative)**:
- 🇹🇼 TW: 使用「量化」
- 🇨🇳 CN: 使用「量化」
- 🇯🇵 JP: 使用「量的」

**「論文」(Paper / Thesis / Dissertation)**:
- Paper = 論文（期刊文章）
- Thesis = 學位論文（碩博士）
- Dissertation = 博士論文（美國用法）
- Context: 依文獻類型使用正確術語

**「引用」vs「參考文獻」(Citation vs Reference)**:
- Citation = 引用（內文標註）
- Reference = 參考文獻（完整書目）
- Bibliography = 書目（與 Reference 同義）

### Citation Style Naming

保持引用格式的英文名稱：

- APA = American Psychological Association
- MLA = Modern Language Association
- Chicago = Chicago Manual of Style
- IEEE = Institute of Electrical and Electronics Engineers
- Vancouver = Vancouver style (醫學領域)

**使用範例**：
```
根據 APA 格式（第 7 版），引用應包含...
According to APA format (7th edition), citations should include...
```

### Academic Degree Abbreviations

學位縮寫保持英文：

- PhD = Doctor of Philosophy (哲學博士)
- MA = Master of Arts (文學碩士)
- MS = Master of Science (理學碩士)
- MBA = Master of Business Administration (企業管理碩士)
- BA = Bachelor of Arts (文學學士)
- BS = Bachelor of Science (理學學士)

**使用範例**：
```
王小明，PhD（哲學博士）
Wang Xiaoming, PhD (Doctor of Philosophy)
```

### Statistical Term Usage

統計術語保持一致性：

```
// ✅ Good - 一致使用英文符號
p < 0.05（顯著）
r = 0.72（相關係數）
M = 4.2, SD = 1.3（平均數與標準差）

// ❌ Bad - 混用中英文
p < 0.05（显著）
相關係數 = 0.72
平均數 = 4.2, 標準差 = 1.3
```

### Thesis Structure Naming

論文章節名稱（依學校規範）：

**英文**:
1. Introduction
2. Literature Review
3. Methodology
4. Results
5. Discussion
6. Conclusion

**繁體中文**（台灣）:
1. 緒論
2. 文獻回顧
3. 研究方法
4. 研究結果
5. 討論
6. 結論

**簡體中文**（中國）:
1. 引言
2. 文献综述
3. 研究方法
4. 研究结果
5. 讨论
6. 结论

### Research Ethics Compliance

涉及人體研究時，必須使用正確術語：

- Informed Consent = 知情同意書（必須有書面同意）
- IRB Approval = 倫理審查委員會核准（必須取得）
- Confidentiality = 保密性（個資保護）
- Anonymity = 匿名性（隱藏身分）

### Authorship Guidelines

依國際慣例，作者順序通常為：

1. **First Author**（第一作者）：主要貢獻者
2. **Co-authors**（共同作者）：依貢獻程度排序
3. **Corresponding Author**（通訊作者）：負責聯繫（標示 *）

**範例**：
```
Wang, X.*, Chen, Y., & Li, Z. (2024). Title of the paper. 
Journal Name, 10(2), 123-145. 
*Corresponding author: wang@example.edu
```

### Abbreviation Guidelines

學術研究常用縮寫：

- et al. = et alii（拉丁文：和其他人）
- e.g. = exempli gratia（拉丁文：例如）
- i.e. = id est（拉丁文：即）
- ibid. = ibidem（拉丁文：同上）
- cf. = confer（拉丁文：參見）
- IRB = Institutional Review Board（倫理審查委員會）
- DOI = Digital Object Identifier（數位物件識別碼）
- ISBN = International Standard Book Number
- ISSN = International Standard Serial Number
