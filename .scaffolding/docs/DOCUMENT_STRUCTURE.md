# DOCUMENT STRUCTURE

**Status**: Active | **Domain**: Academic  
**Related Modules**: ACADEMIC_WRITING, RESEARCH_ORGANIZATION, CITATION_MANAGEMENT

## Purpose

This module defines standards for structuring academic and professional documents, including research papers, theses, dissertations, reports, and technical documentation. It covers section organization, heading hierarchies, and document flow.

## When to Use This Module

- Writing research papers or articles
- Structuring thesis or dissertation chapters
- Creating technical reports
- Organizing white papers or position papers
- Planning document outlines

---

## 1. Research Paper Structure (IMRaD)

### 1.1 IMRaD Format

**I**ntroduction - **M**ethods - **R**esults - **a**nd **D**iscussion

```
Title
Abstract
Keywords

1. Introduction
   1.1 Background
   1.2 Problem Statement
   1.3 Research Question/Hypothesis
   1.4 Objectives
   1.5 Significance

2. Literature Review (or part of Introduction)
   2.1 Theoretical Framework
   2.2 Empirical Evidence
   2.3 Research Gaps

3. Methods
   3.1 Research Design
   3.2 Participants/Sample
   3.3 Materials/Instruments
   3.4 Procedure
   3.5 Data Analysis

4. Results
   4.1 Descriptive Statistics
   4.2 Main Findings
   4.3 Additional Analyses

5. Discussion
   5.1 Summary of Findings
   5.2 Interpretation
   5.3 Implications
   5.4 Limitations
   5.5 Future Research

6. Conclusion

References
Appendices
```

## 3. Heading Hierarchy

### 3.1 APA Style Headings

```
Level 1: Centered, Bold, Title Case
Methods

Level 2: Flush Left, Bold, Title Case
Participants

Level 3: Flush Left, Bold Italic, Title Case
Demographic Characteristics

Level 4: Indented, Bold, Title Case, Ending with Period. Text begins...
Age Distribution. The participants ranged from...

Level 5: Indented, Bold Italic, Title Case, Ending with Period. Text begins...
Age Groups. Participants were categorized into...
```

### 3.2 Numbered Headings

```
1. Introduction
  1.1 Background
    1.1.1 Historical Context
    1.1.2 Current Situation
  1.2 Problem Statement

2. Literature Review
  2.1 Theoretical Framework
  2.2 Empirical Evidence
```

## 4. Section Guidelines

### 4.1 Abstract

**Length**: 150-300 words  
**Tense**: Past tense for completed research

```
Structure:
1. Background (1-2 sentences)
2. Objective (1 sentence)
3. Methods (2-3 sentences)
4. Results (2-3 sentences)
5. Conclusion (1-2 sentences)

Example:
"Type 2 diabetes affects millions worldwide, with diet playing 
a crucial role in management. This study examined the effects 
of intermittent fasting on glycemic control in adults with type 
2 diabetes. A randomized controlled trial was conducted with 120 
participants assigned to either 16:8 time-restricted eating or 
standard dietary advice for 12 weeks. The intermittent fasting 
group showed significantly greater reductions in HbA1c (-0.8% vs 
-0.2%, p<0.001) and fasting glucose. Intermittent fasting appears 
to be an effective dietary strategy for improving glycemic control 
in type 2 diabetes."
```

### 4.2 Introduction

**Purpose**: Establish context, identify gap, state research question

```
Funnel Structure:
[BROAD] General topic introduction
  ↓
[NARROWER] Specific problem/gap
  ↓
[SPECIFIC] Research question/hypothesis

Length: 10-15% of total paper
```

### 4.3 Literature Review

**Organization Options**:
1. **Chronological**: Historical development
2. **Thematic**: Organized by themes/topics
3. **Methodological**: Grouped by research methods
4. **Theoretical**: Organized by theoretical frameworks

```
✅ GOOD: Synthesis and critique
"Three main theories explain X: Theory A emphasizes..., 
Theory B focuses on..., while Theory C proposes... Despite 
their contributions, these theories fail to account for..."

❌ BAD: Just listing studies
"Smith (2020) found X. Jones (2021) found Y. Brown (2022) 
found Z."
```

### 4.4 Methods

**Subsections**:
```
Research Design
- Type of study (experimental, observational, qualitative, etc.)
- Justification for design choice

Participants/Sample
- Inclusion/exclusion criteria
- Recruitment method
- Sample size and power analysis
- Demographic characteristics

Materials/Instruments
- Description of tools/measures
- Validity and reliability
- Procedures for development (if new instrument)

Procedure
- Step-by-step process
- Timeline
- Ethical considerations

Data Analysis
- Statistical tests used
- Software
- Significance level
```

**Criteria**: Enough detail for replication

### 4.5 Results

**Organization**:
```
1. Preliminary Analyses
   - Data screening
   - Assumption checks
   - Missing data handling

2. Main Analyses
   - Hypothesis 1
   - Hypothesis 2
   - Hypothesis 3

3. Additional Analyses
   - Post-hoc tests
   - Exploratory findings
```

**Guidelines**:
- Report facts, not interpretations
- Use tables/figures for complex data
- Report all relevant statistics
- State effect sizes and confidence intervals

### 4.6 Discussion

**Structure**:
```
Opening: Brief summary of main findings

Body:
1. Interpretation of each major finding
2. Comparison with previous research
3. Theoretical implications
4. Practical implications

Limitations:
- Sample limitations
- Measurement limitations
- Design limitations

Future Research:
- Specific suggestions based on limitations
- New questions raised

Closing: Overall conclusion
```

## 5. Figures and Tables

### 5.1 Figure Placement

```
Option 1: Embedded in text (after first mention)
"As shown in Figure 1, there was a significant effect..."

[Figure 1 here]

Option 2: At end of document (journal submission)
Text: "...significant effect (see Figure 1)."
Figures Section: [All figures at end]
```

### 5.2 Figure Captions

```
Figure 1. Mean HbA1c levels by group over 12 weeks. Error 
bars represent standard error. IF = intermittent fasting; 
SD = standard diet. *p < 0.05, **p < 0.01.
```

### 5.3 Table Format

```
Table 1
Participant Characteristics by Group

                        IF Group    Control     p-value
                        (n = 60)    (n = 60)
Age (years)            52.3 (8.1)   51.8 (7.9)  0.72
Female, n (%)          35 (58%)     32 (53%)    0.59
BMI (kg/m²)            31.2 (4.5)   30.8 (4.3)  0.61
HbA1c (%)              8.1 (1.2)    8.0 (1.1)   0.68

Note. Values are mean (SD) unless otherwise indicated.
IF = intermittent fasting; BMI = body mass index.
```

### 6.1 What to Include


## 8. Document Flow

### 8.1 Transitions Between Sections

```
✅ GOOD: Smooth transitions
"Having established the theoretical framework, the next section 
describes the methodology used to test these hypotheses."

"The results presented above demonstrate significant effects. 
The following section interprets these findings in light of 
existing literature."

❌ BAD: Abrupt section changes
[End of Literature Review]
Methods
[Start describing methods]
```

### 8.2 Paragraph Structure

```
Topic Sentence: Main idea of paragraph
Supporting Sentences: Evidence, examples, explanations
Concluding/Transition Sentence: Link to next paragraph

Example:
"Intermittent fasting has shown promise for metabolic health. 
[TOPIC] Several studies have reported improvements in insulin 
sensitivity (Smith et al., 2020) and weight loss (Jones et al., 
2021). [SUPPORT] These findings suggest that intermittent 
fasting may be particularly beneficial for individuals with 
metabolic disorders. [CONCLUSION/TRANSITION]"
```

## 9. Style Guides

### 9.1 APA Style (7th Edition)

- Psychology, education, social sciences
- Author-date citations: (Smith, 2020)
- Reference list at end

### 9.2 MLA Style (9th Edition)

- Humanities, literature
- Parenthetical citations: (Smith 23)
- Works Cited at end

### 9.3 Chicago Style (17th Edition)

- History, arts
- Footnotes or endnotes
- Bibliography at end

### 9.4 IEEE Style

- Engineering, computer science
- Numbered citations: [1]
- References at end

## Anti-Patterns

### ❌ Unbalanced Sections
Introduction (10 pages), Methods (2 pages), Results (1 page), Discussion (8 pages)

**Solution**: Aim for proportional sections.

### ❌ Inconsistent Heading Levels
Skipping levels (Level 1 → Level 3)

**Solution**: Follow hierarchical structure.

### ❌ Results in Discussion
Presenting new data in discussion section

**Solution**: All results go in Results section; Discussion interprets them.

### ❌ Methods in Introduction
Detailed methodology in introduction

**Solution**: Brief overview in intro, full details in Methods.
